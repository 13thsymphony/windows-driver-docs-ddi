# _WDI_TYPE_NONCE structure
The WDI_TYPE_NONCE structure defines the SNonce or ANonce (802.11r).

## Syntax
````
typedef struct _WDI_TYPE_NONCE {
  UINT8 Nonce[32];
} WDI_TYPE_NONCE, *PWDI_TYPE_NONCE;
````

## Members


`Nonce`

The SNonce or ANonce.
