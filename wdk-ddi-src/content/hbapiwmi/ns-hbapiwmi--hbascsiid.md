# HBAScsiID structure



## -description
<p>The HBAScsiID structure contains information generated by the operating system that uniquely identifies a logical unit.</p>


## -syntax

````
typedef struct _HBAScsiID {
  ULONG  ScsiBusNumber;
  ULONG  ScsiTargetNumber;
  ULONG  ScsiOSLun;
  USHORT OSDeviceName[257];
} HBAScsiID, *PHBAScsiID;
````


## -struct-fields
<dl>

### -field ScsiBusNumber

<dd>
<p>Contains the bus number. </p>
</dd>

### -field ScsiTargetNumber

<dd>
<p>Contains the target device number. </p>
</dd>

### -field ScsiOSLun

<dd>
<p>Contains the logical unit number (LUN). </p>
</dd>

### -field OSDeviceName

<dd>
<p>Contains a nonpersistent target name such as "\Device\HarddiskVolume1".</p>
</dd>
</dl>

## -remarks
<p>For more information about how this structure is defined, see the discussion of the HBA_ScsiID structure in the T11 committee's <i>Fibre Channel HBA API</i> specification.</p>
