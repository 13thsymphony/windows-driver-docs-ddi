# DXGKDDI_CREATEPERIODICFRAMENOTIFICATION function
Used to create a periodic frame notification.

## Syntax

```
DXGKDDI_CREATEPERIODICFRAMENOTIFICATION DxgkddiCreateperiodicframenotification;

NTSTATUS DxgkddiCreateperiodicframenotification(
  INOUT_PDXGKARG_CREATEPERIODICFRAMENOTIFICATION pCreatePeriodicFrameNotification
)
{...}
```

## Parameters

`pCreatePeriodicFrameNotification`

A structure of type <i>PDXGKARG_CREATEPERIODICFRAMENOTIFICATION</i> containing the arguments needed to create a periodic frame notification.
