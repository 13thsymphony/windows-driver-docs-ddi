# _DXGK_QUERYDISPLAYIDOUT structure
Used to query a display ID.

## Syntax
````
typedef struct _DXGK_QUERYDISPLAYIDOUT {
  UINT Length;
  BYTE *pDescriptor;
} DXGK_QUERYDISPLAYIDOUT;
````

## Members


`Length`

The length of the Display ID.

`pDescriptor`

A pointer to the descriptor of the ID.
