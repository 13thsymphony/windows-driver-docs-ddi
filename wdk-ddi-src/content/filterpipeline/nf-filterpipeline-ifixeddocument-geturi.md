# IFixedDocument::GetUri method
The <b>GetUri</b> method gets the URI of the fixed document.

## Syntax

```
HRESULT GetUri(
  BSTR *uri
);
```

## Parameters

`uri`

The URI of the fixed document.


## Return Value

<b>GetUri</b> returns an <b>HRESULT</b> value.
