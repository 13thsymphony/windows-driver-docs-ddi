# IFixedPage::SetPagePart method
The <b>SetPagePart</b> method associates a new part with the page.

## Syntax

```
HRESULT SetPagePart(
  IUnknown *pUnk
);
```

## Parameters

`pUnk`

A pointer to the new part.


## Return Value

<b>SetPagePart</b> returns an <b>HRESULT</b> value.
