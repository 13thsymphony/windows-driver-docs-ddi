# StorPortReadPortUshort function
The <b>StorPortReadPortUshort</b> routine reads a value from a specified port address.

## Syntax

````
STORPORT_API USHORT StorPortReadPortUshort(
  _In_ PVOID   HwDeviceExtension,
  _In_ PUSHORT Port
);
````

## Parameters

`HwDeviceExtension`



`Port`




## Return Value

None
