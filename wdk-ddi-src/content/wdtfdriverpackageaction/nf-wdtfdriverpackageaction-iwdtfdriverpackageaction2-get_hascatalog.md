# get_HasCatalog method
Gets a value that indicates whether the driver package has a catalog.

This property is read-only.

## Syntax

````
HRESULT get_HasCatalog(
  [out, retval] VARIANT_BOOL *pbVal
);
````

## Parameters

`pbVal`




## Return Value

None
