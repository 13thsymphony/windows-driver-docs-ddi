# D3DKMTOutputDuplReleaseFrame function



## -description
<p>Indicates that the driver has finished processing the duplicated desktop image.</p>


## -syntax

````
EXTERN_C _Check_return_ NTSTATUS APIENTRY D3DKMTOutputDuplReleaseFrame(
  _Inout_ D3DKMT_OUTPUTDUPL_RELEASE_FRAME *pData
);
````


## -parameters
<dl>

### -param <i>pData</i> [in, out]

<dd>
<p>A pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/hh406533">D3DKMT_OUTPUTDUPL_RELEASE_FRAME</a> structure that defines the duplicated desktop image that is to be released.</p>
</dd>
</dl>

## -returns
<p>Returns one of the following values:</p><dl>
<dt><b>STATUS_SUCCESS</b></dt>
</dl><p>The function performed successfully.</p><dl>
<dt><b>STATUS_INVALID_PARAMETER </b></dt>
</dl><p>
         Parameters were validated and determined to be incorrect.</p><dl>
<dt><b>STATUS_NOT_FOUND</b></dt>
</dl><p>The context of the process could not be found.</p>

<p> </p>

<p>This function might also return other NTSTATUS values.</p>
