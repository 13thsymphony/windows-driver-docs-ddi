# PFND3D12DDI_CALCPRIVATEVIDEODECODERHEAPSIZE_0033 callback



## -description
<p>Used to calculate the size of a video decoder heap.</p>


## -prototype

````
SIZE_T APIENTRY* PFND3D12DDI_CALCPRIVATEVIDEODECODERHEAPSIZE_0033(
             D3D12DDI_HDEVICE                           hDrvDevice,
  _In_ const D3D12DDIARG_CREATE_VIDEO_DECODER_HEAP_0033 *pArgs
);
````


## -parameters
<dl>

### -param <i>hDrvDevice</i> 

<dd>
<p>The hardware device being processed.</p>
</dd>

### -param <i>pArgs</i> [in]

<dd>
<p>The arguments used to create a video decoder heap.</p>
</dd>
</dl>

## -returns
<p>Returns the size of the heap in bytes.</p>
