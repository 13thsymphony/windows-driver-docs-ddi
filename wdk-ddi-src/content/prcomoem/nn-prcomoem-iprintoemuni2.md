# IPrintOemUni2 interface

This section describes the methods defined for the IPrintOemUni2 COM interface.

In addition to the methods that belong to the IPrintOemUni2 COM interface, this interface includes all of the methods that belong to the <a href="..\prcomoem\nn-prcomoem-iprintoemuni.md">IPrintOemUni</a> COM interface.

## Methods

<p>The <b>IPrintOemUni2</b> interface has these methods.</p>

| Method | Description |
| ---- |:---- |
| [IPrintOemUni2::GetImplementedMethod](nf-prcomoem-iprintoemuni2-getimplementedmethod.md) | The IPrintOemUni2::GetImplementedMethod method is used by Unidrv to determine which IPrintOemUni2 interface methods a rendering plug-in has implemented. |
| [IPrintOemUni2::WritePrinter](nf-prcomoem-iprintoemuni2-writeprinter.md) | The IPrintOemUni2::WritePrinter method, if supported, enables a rendering plug-in to capture all output data generated by a Unidrv driver. |
