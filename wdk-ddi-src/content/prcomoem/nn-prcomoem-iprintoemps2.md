# IPrintOemPS2 interface

This section describes the methods defined for the <b>IPrintOemPS2</b> COM interface. In addition to these methods, this interface includes all of the methods defined in the <a href="https://msdn.microsoft.com/14c545b7-8080-424f-9164-f97ef8a1acc2">IPrintOemPS</a> COM interface.

## Methods

<p>The <b>IPrintOemPS2</b> interface has these methods.</p>

| Method | Description |
| ---- |:---- |
| [IPrintOemPS2::GetPDEVAdjustment](nf-prcomoem-iprintoemps2-getpdevadjustment.md) | The IPrintOemPS2::GetPDEVAdjustment method enables a plug-in to override specific PDEV settings. |
| [IPrintOemPS2::WritePrinter](nf-prcomoem-iprintoemps2-writeprinter.md) | The IPrintOemPS2::WritePrinter method, if supported, enables a rendering plug-in to capture all output data generated by a Postscript driver. |
