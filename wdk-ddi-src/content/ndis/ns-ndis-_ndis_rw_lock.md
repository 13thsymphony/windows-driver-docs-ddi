# _NDIS_RW_LOCK structure
The <b>NDIS_RW_LOCK</b> structure defines the attributes of a read/write lock. This structure is opaque to
   NDIS drivers.

## Syntax
````
typedef struct _NDIS_RW_LOCK {
  ;      // Reserved for NDIS.
} NDIS_RW_LOCK, *PNDIS_RW_LOCK;
````
