# _NDIS_RW_LOCK_REFCOUNT structure
The <b>NDIS_RW_LOCK_REFCOUNT</b> union defines attributes of a read/write lock. This union is opaque to NDIS
   drivers.

## Syntax
```
typedef struct _NDIS_RW_LOCK_REFCOUNT {
  ULONG RefCount;
  UCHAR cacheLine[16];
} NDIS_RW_LOCK_REFCOUNT;
```
