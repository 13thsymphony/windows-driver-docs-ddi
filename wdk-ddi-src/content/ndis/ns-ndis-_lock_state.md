# _LOCK_STATE structure
The <b>LOCK_STATE</b> structure tracks the state of a read/write lock. This structure is opaque to NDIS
   drivers.

## Syntax
````
typedef struct _LOCK_STATE {
  ;      // Reserved for NDIS.
} LOCK_STATE, *PLOCK_STATE;
````
