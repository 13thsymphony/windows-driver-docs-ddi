# smcnt.h header



smcnt.h contains the following programming interfaces:





## Functions
| Title | Description |
| ---- |:---- |
| [ExAllocatePool](nf-smcnt-exallocatepool.md) | The ExAllocatePool routine is obsolete, and is exported only for existing binaries. Use ExAllocatePoolWithTag instead.ExAllocatePool allocates pool memory of the specified type and returns a pointer to the allocated block. |



## Structures
| Title | Description |
| ---- |:---- |
| [_OS_DEP_DATA](ns-smcnt-_os_dep_data.md) | The OS_DEP_DATA structure defines the data that is stored in the OsData member of the SMARTCARD_EXTENSION structure, which holds smart card information that is specific to the operating system. |
