# IoThreadToProcess function



## -description
<p>The <b>IoThreadToProcess</b> routine returns a pointer to the process for the specified thread.</p>


## -syntax

````
PEPROCESS IoThreadToProcess(
  _In_ PETHREAD Thread
);
````


## -parameters
<dl>

### -param <i>Thread</i> [in]

<dd>
<p>Thread whose process is to be returned.</p>
</dd>
</dl>

## -returns
<p><b>IoThreadToProcess</b> returns a pointer to the thread's process.</p>

<p>For more information about using system threads and managing synchronization within a nonarbitrary thread context, see <a href="https://msdn.microsoft.com/fbd8aadd-5a24-48c9-9865-80cc7dc97316">Driver Threads, Dispatcher Objects, and Resources</a>. </p>
