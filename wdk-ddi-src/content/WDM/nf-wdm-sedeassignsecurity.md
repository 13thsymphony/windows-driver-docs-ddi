# SeDeassignSecurity function



## -description
<p>The  
   <b>SeDeassignSecurity</b> routine deallocates the memory associated with a security descriptor that was assigned using <b>SeAssignSecurity</b>.</p>


## -syntax

````
NTSTATUS SeDeassignSecurity(
  _Inout_ PSECURITY_DESCRIPTOR *SecurityDescriptor
);
````


## -parameters
<dl>

### -param <i>SecurityDescriptor</i> [in, out]

<dd>
<p>Pointer to the buffered <a href="https://msdn.microsoft.com/library/windows/hardware/ff563689">SECURITY_DESCRIPTOR</a> being released.</p>
</dd>
</dl>

## -returns
<p>If the deallocation succeeds, <b>SeDeassignSecurity</b> returns STATUS_SUCCESS.</p>
