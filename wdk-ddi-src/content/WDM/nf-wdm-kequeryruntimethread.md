# KeQueryRuntimeThread function



## -description
<p>The <b>KeQueryRuntimeThread</b> routine reports the accumulated kernel-mode and user-mode run time of a thread, in clock ticks.</p>


## -syntax

````
ULONG KeQueryRuntimeThread(
  _In_  PKTHREAD Thread,
  _Out_ PULONG   UserTime
);
````


## -parameters
<dl>

### -param <i>Thread</i> [in]

<dd>
<p>Pointer to a dispatcher object of type KTHREAD.</p>
</dd>

### -param <i>UserTime</i> [out]

<dd>
<p>Pointer to the memory location where <b>KeQueryRuntimeThread</b> returns the accumulated user-mode run time of the current thread, in clock ticks.</p>
</dd>
</dl>

## -returns
<p><b>KeQueryRuntimeThread</b> returns the accumulated kernel-mode run time of the current thread, in clock ticks.</p>
