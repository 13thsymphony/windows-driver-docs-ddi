# _LPIT structure
Defines an LPI ACPI table.

## Syntax
````
typedef struct _LPIT {
  DESCRIPTION_HEADER   Header;
  LPI_STATE_DESCRIPTOR LpiStates[ANYSIZE_ARRAY];
} LPIT, *PLPIT;
````

## Members

        
            `Header`

            A header.
        
            `LpiStates`

            An array of states.
