# IStiUSD::GetLastError method



## -description
<p>The <b>IStiUSD::GetLastError</b> method returns the last known error associated with a still image device.</p>


## -syntax

````
HRESULT GetLastError(
   LPDWORD pdwLastDeviceError
);
````


## -parameters
<dl>

### -param <i>pdwLastDeviceError</i> 

<dd>
<p>Caller-supplied pointer to a buffer in which the error code will be stored.</p>
</dd>
</dl>

## -returns
<p>If the operation succeeds, the method returns S_OK. Otherwise, it returns one of the STIERR-prefixed error codes defined in <i>stierr.h</i>.</p>
