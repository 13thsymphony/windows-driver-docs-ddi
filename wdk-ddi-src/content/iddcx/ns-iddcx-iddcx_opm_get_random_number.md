# IDDCX_OPM_GET_RANDOM_NUMBER structure
Gives the OPM random number generated by the driver.

## Syntax
````
typedef struct IDDCX_OPM_GET_RANDOM_NUMBER {
  UINT              Size;
  OPM_RANDOM_NUMBER RandomNumber;
} IDDCX_OPM_GET_RANDOM_NUMBER, *IDDCX_OPM_GET_RANDOM_NUMBER;
````

## Members

        
            `RandomNumber`

            The random number the driver generated.
        
            `Size`

            Total size of the structure.
