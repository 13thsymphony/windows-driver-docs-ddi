# WIAS_HRESULT function
The WIAS_HRESULT macro writes a diagnostic message to the <i>Wiatrace.log</i> file.

## Syntax

````
HRESULT WIAS_HRESULT(
   HINSTANCE hInstance,
   HRESULT   hr
);
````

`x`

TBD


## Return Value

None

## Remarks

This macro is the recommended way to output HRESULTS on Windows Vista.
