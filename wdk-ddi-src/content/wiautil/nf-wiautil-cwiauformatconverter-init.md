# CWiauFormatConverter::Init method



## -description
The <b>CWiauFormatConverter::Init</b> method initializes the <b>CWiauFormatConverter</b> class and GDI+ for converting images. This method should be called only once.



## -syntax

````
HRESULT Init();
````
