# CWiauDbgFn method
The <b>CWiauDbgFn::CWiauDbgFn</b> method is used for tracing when a function or method is entered.

## Syntax

````
void CWiauDbgFn(
   LPCSTR   fname
);
````

## Parameters

`fname`

Specifies the name of the function or method being entered.


## Return Value

This method does not return a value.
