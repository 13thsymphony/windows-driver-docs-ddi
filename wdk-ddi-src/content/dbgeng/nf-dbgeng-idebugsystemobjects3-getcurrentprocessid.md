# GetCurrentProcessId method
The <b>GetCurrentProcessId</b> method returns the engine process ID for the current process.

## Syntax

````
HRESULT GetCurrentProcessId(
  [out]Â PULONG Id
);
````

## Parameters

`Id`

Receives the engine process ID for the current process.
