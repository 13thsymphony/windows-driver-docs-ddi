# IDebugSymbols2 interface



## -description

## -inheritance
The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IDebugSymbols2</b> interface inherits from <a href="..\dbgeng\nn-dbgeng-idebugsymbols.md">IDebugSymbols</a>. <b>IDebugSymbols2</b> also has these types of members:

The <b>IDebugSymbols2</b> interface has these methods.

Turns on some type formatting options for output generated by the engine.

Returns the name of the specified constant.

Returns the name of a field within a structure.

Returns the name of the specified module.

Returns version information for the specified module.

Returns the type formatting options for output generated by the engine.

Turns off some type formatting options for output generated by the engine.

Changes the type formatting options for output generated by the engine.
