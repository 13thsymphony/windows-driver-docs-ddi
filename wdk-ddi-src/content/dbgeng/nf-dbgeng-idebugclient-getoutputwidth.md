# IDebugClient::GetOutputWidth method



## -description
<p>Gets the width of an output line for
    commands that produce formatted output. </p>


## -syntax

````
HRESULT GetOutputWidth(
  [out] PULONG Columns
);
````


## -parameters
<dl>

### -param Columns [out]

<dd>
<p>The number of columns in the output.</p>
</dd>
</dl>

## -returns
<p>This method may also return error values.  See <a href="https://msdn.microsoft.com/713f3ee2-2f5b-415e-9908-90f5ae428b43">Return Values</a> for more details.</p><dl>
<dt><b>S_OK</b></dt>
</dl><p>The method was successful.</p>

<p> </p>

## -remarks
<p>This setting is a suggestion that can be overridden by other settings.</p>
