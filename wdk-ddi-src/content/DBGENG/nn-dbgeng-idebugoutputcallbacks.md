# IDebugOutputCallbacks interface



## -description

## -inheritance
<p>The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IDebugOutputCallbacks</b> interface inherits from the <a href="com.iunknown" xmlns:loc="http://microsoft.com/wdcml/l10n"><b>IUnknown</b></a> interface. <b>IDebugOutputCallbacks</b> also has these types of members:</p>

<p>The <b>IDebugOutputCallbacks</b> interface has these methods.</p>
