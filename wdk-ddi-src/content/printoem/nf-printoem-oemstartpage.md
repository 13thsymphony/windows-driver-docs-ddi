# OEMStartPage function
The <code>OEMStartPage</code> function is called by GDI when it is ready to start sending the contents of a physical page to the driver for rendering.

## Syntax

````
BOOL APIENTRY OEMStartPage(
   SURFOBJ *pso
);
````

## Parameters

`pso`




## Return Value

None
