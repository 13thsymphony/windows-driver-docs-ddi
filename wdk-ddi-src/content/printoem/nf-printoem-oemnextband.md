# OEMNextBand function
The <code>OEMNextBand</code> function is called by GDI when it has finished drawing a band for a physical page, so that the driver can send the band to the printer.

## Syntax

```
BOOL OEMNextBand(
  SURFOBJ *pso,
  POINTL  *pptl
);
```

## Parameters

`pso`



`pptl`




## Return Value

None
