# OEMIcmDeleteColorTransform function
The <code>OEMIcmDeleteColorTransform</code> function deletes the specified color transform.

## Syntax

```
BOOL OEMIcmDeleteColorTransform(
  DHPDEV dhpdev,
  HANDLE hcmXform
);
```

## Parameters

`dhpdev`



`hcmXform`




## Return Value

None
