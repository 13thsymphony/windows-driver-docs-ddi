# OEMQueryFontData function
The <code>OEMQueryFontData</code> function retrieves information about a realized font.

## Syntax

```
LONG OEMQueryFontData(
  DHPDEV    dhpdev,
  FONTOBJ   *pfo,
  ULONG     iMode,
  HGLYPH    hg,
  GLYPHDATA *pgd,
  PVOID     pv,
  ULONG     cjSize
);
```

## Parameters

`dhpdev`



`pfo`



`iMode`



`hg`



`pgd`



`pv`



`cjSize`




## Return Value

None
