# OEMQueryFontData function
The <code>OEMQueryFontData</code> function retrieves information about a realized font.

## Syntax

````
LONG APIENTRY OEMQueryFontData(
        DHPDEV                       dhpdev,
        FONTOBJ                      *pfo,
        ULONG                        iMode,
        HGLYPH                       hg,
        GLYPHDATA                    *pgd,
  _Out_ _writes_bytes_(cjSize) PVOID pv,
        ULONG                        cjSize
);
````

## Parameters

`dhpdev`



`pfo`



`iMode`



`hg`



`pgd`



`pv`



`cjSize`




## Return Value

None
