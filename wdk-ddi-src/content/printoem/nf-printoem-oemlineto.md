# OEMLineTo function
The <code>OEMLineTo</code> function draws a single, solid, integer-only cosmetic line.

## Syntax

````
BOOL APIENTRY OEMLineTo(
   SURFOBJ  *pso,
   CLIPOBJ  *pco,
   BRUSHOBJ *pbo,
   LONG     x1,
   LONG     y1,
   LONG     x2,
   LONG     y2,
   RECTL    *prclBounds,
   MIX      mix
);
````

## Parameters

`pso`



`pco`



`pbo`



`x1`



`y1`



`x2`



`y2`



`prclBounds`



`mix`




## Return Value

None
