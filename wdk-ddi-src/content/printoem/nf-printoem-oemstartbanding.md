# OEMStartBanding function
The <code>OEMStartBanding</code> function is called by GDI when it is ready to start sending bands of a physical page to the driver for rendering.

## Syntax

````
BOOL APIENTRY OEMStartBanding(
   SURFOBJ *pso,
   POINTL  *pptl
);
````

## Parameters

`pso`



`pptl`




## Return Value

None
