# OEMStretchBlt function
The <code>OEMStretchBlt</code> function provides stretching bit-block transfer capabilities between any combination of device-managed and GDI-managed surfaces.

## Syntax

```
BOOL OEMStretchBlt(
  SURFOBJ         *psoDest,
  SURFOBJ         *psoSrc,
  SURFOBJ         *psoMask,
  CLIPOBJ         *pco,
  XLATEOBJ        *pxlo,
  COLORADJUSTMENT *pca,
  POINTL          *pptlHTOrg,
  RECTL           *prclDest,
  RECTL           *prclSrc,
  POINTL          *pptlMask,
  ULONG           iMode
);
```

`psoMask`



`pco`
