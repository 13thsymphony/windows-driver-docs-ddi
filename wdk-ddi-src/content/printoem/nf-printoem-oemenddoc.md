# OEMEndDoc function
The <code>OEMEndDoc</code> function is called by the GDI when it has finished sending a document to the driver for rendering.

## Syntax

```
BOOL OEMEndDoc(
  SURFOBJ *pso,
  FLONG   fl
);
```

## Parameters

`pso`



`fl`




## Return Value

None
