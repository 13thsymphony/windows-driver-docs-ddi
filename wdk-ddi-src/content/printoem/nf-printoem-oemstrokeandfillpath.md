# OEMStrokeAndFillPath function
The <code>OEMStrokeAndFillPath</code> function concurrently strokes and fills a path.

## Syntax

```
BOOL OEMStrokeAndFillPath(
  SURFOBJ   *pso,
  PATHOBJ   *ppo,
  CLIPOBJ   *pco,
  XFORMOBJ  *pxo,
  BRUSHOBJ  *pboStroke,
  LINEATTRS *plineattrs,
  BRUSHOBJ  *pboFill,
  POINTL    *pptlBrushOrg,
  MIX       mixFill,
  FLONG     flOptions
);
```

`ppo`



`pco`



`pxo`



`pboStroke`
