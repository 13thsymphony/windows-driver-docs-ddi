# DOT11_KEY_ALGO_BIP structure


## Syntax
````
typedef struct _DOT11_KEY_ALGO_BIP {
  UCHAR ucIPN[6];
  ULONG ulBIPKeyLength;
  UCHAR ucBIPKey[1];
} DOT11_KEY_ALGO_BIP, *PDOT11_KEY_ALGO_BIP;
````

## Members

        
            `ulBIPKeyLength`

            The length, in bytes, of the BIP key material in <b>ucBIPKey</b> array.
