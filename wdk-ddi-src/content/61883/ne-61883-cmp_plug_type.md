# CMP_PLUG_TYPE enumeration



## -description
This enumeration specifies the type of a plug.



## -syntax

````
typedef enum  { 
  CMP_PlugOut  = 0,
  CMP_PlugIn
} CMP_PLUG_TYPE;
````


## -enum-fields

### -field CMP_PlugOut

An out plug.


### -field CMP_PlugIn

An in plug.
