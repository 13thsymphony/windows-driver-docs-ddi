# CMP_PLUG_TYPE Enumeration
This enumeration specifies the type of a plug.

## Syntax
````
typedef enum  { 
  CMP_PlugOut  = 0,
  CMP_PlugIn
} CMP_PLUG_TYPE;
````

## Constants

<table>
            
                <tr>
                    <td>CMP_PlugIn</td>
                    <td>An in plug.</td>
                </tr>
            
                <tr>
                    <td>CMP_PlugOut</td>
                    <td>An out plug.</td>
                </tr>
