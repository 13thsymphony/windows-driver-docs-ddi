# CMP_PLUG_LOCATION Enumeration
This enumeration specifies the location of a plug.

## Syntax
````
typedef enum  { 
  CMP_PlugLocal   = 0,
  CMP_PlugRemote
} CMP_PLUG_LOCATION;
````

## Constants

<table>
            
                <tr>
                    <td>CMP_PlugLocal</td>
                    <td>A local plug.</td>
                </tr>
            
                <tr>
                    <td>CMP_PlugRemote</td>
                    <td>A remote plug.</td>
