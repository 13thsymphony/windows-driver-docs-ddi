# NfcCxDeviceInitialize function



## -description
<p>Called by the client driver after a WDF device has been created during the AddDevice routine.</p>


## -syntax

````
NTSTATUS  NfcCxDeviceInitialize(
   WDFDEVICE Device
);
````


## -parameters
<dl>

### -param <i>Device</i> 

<dd>
<p>A handle to a framework device object.</p>
</dd>
</dl>

## -returns
<p>If the operation succeeds, the function returns STATUS_SUCCESS.</p>
