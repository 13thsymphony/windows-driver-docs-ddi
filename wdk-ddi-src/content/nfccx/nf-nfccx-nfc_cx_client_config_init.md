# NFC_CX_CLIENT_CONFIG_INIT function



## -description
The <b>NFC_CX_CLIENT_CONFIG_INIT</b> function initializes the <a href="nfpdrivers.nfc_cx_client_config">NFC_CX_CLIENT_CONFIG</a> structure.


## -syntax

````
FORCEINLINE VOID  NFC_CX_CLIENT_CONFIG_INIT(
  _Out_ PNFC_CX_CLIENT_CONFIG Config,
  _In_  NFC_CX_TRANSPORT_TYPE BusType
);
````


## -parameters

### -param Config [out]

A pointer to an <a href="nfpdrivers.nfc_cx_client_config">NFC_CX_CLIENT_CONFIG</a> structure. 

### -param BusType [in]

An <a href="nfpdrivers.nfc_cx_transport_type">NFC_CX_TRANSPORT_TYPE</a> enumeration. 

## -returns
This function does not return a value.
