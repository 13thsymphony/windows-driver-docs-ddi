# VideoPortReadPortUchar function



## -description
<p>The <b>VideoPortReadPortUchar</b> function reads a byte from a mapped I/O port.</p>


## -syntax

````
UCHAR VideoPortReadPortUchar(
   PUCHAR Port
);
````


## -parameters
<dl>

### -param Port 

<dd>
<p>Pointer to the port. The given <i>Port</i> must be in a mapped I/O-space range returned by <a href="..\video\nf-video-videoportgetdevicebase.md">VideoPortGetDeviceBase</a>.</p>
</dd>
</dl>

## -returns
<p><b>VideoPortReadPortUchar</b> returns the byte read from the adapter.</p>

## -remarks
<p>A miniport driver's <a href="..\video\nc-video-pvideo-hw-interrupt.md">HwVidInterrupt</a> or <a href="..\video\nc-video-pminiport-synchronize-routine.md">HwVidSynchronizeExecutionCallback</a> function can call <b>VideoPortReadPortUchar</b>.</p>

<p>Callers of <b>VideoPortReadPortUchar</b> can be running at any IRQL, provided that the memory pointed to by the <i>Port</i> parameter is resident, mapped device memory.</p>
