# ExtRemoteData::Write method



## -description
<p>The <b>Write</b> method writes the data cached by the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object to the region of memory on the target, represented by this object.</p>


## -syntax

````
void Write();
````


## -parameters


## -returns
<p>This method does not return a value.</p>

<p>This method does not return a value.</p>

<p>This method does not return a value.</p>
