# ExtRemoteData::GetW32Bool method



## -description
<p>The <b>GetW32Bool</b> method returns a BOOL version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object, which represents the contents of the target's memory.</p>


## -syntax

````
BOOL GetW32Bool();
````

## -returns
