# ExtRemoteTyped::operator* method



## -description
<p>The <b>operator*</b> overloaded operator returns the typed data that is pointed to by the typed data represented by this object.</p>


## -syntax

````
ExtRemoteData operator*();
````


## -parameters


## -returns
<p>The <b>operator*</b> operator returns a new <b>ExtRemoteData</b> object that represents the typed data pointed to by the typed data represented by this object.</p>

<p>The <b>operator*</b> operator returns a new <b>ExtRemoteData</b> object that represents the typed data pointed to by the typed data represented by this object.</p>

<p>The <b>operator*</b> operator returns a new <b>ExtRemoteData</b> object that represents the typed data pointed to by the typed data represented by this object.</p>

## -remarks
<p>If the typed data represented by this object is an array, the first element in the array is returned.</p>
