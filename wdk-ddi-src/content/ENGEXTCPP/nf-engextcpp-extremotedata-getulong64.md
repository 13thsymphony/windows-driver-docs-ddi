# ExtRemoteData::GetUlong64 method



## -description
<p>The <b>GetUlong64</b> method returns a ULONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object, which represents the contents of the target's memory.</p>


## -syntax

````
ULONG64 GetUlong64();
````


## -parameters


## -returns
<p><b>GetUlong64</b> returns the ULONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetUlong64</b> returns the ULONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetUlong64</b> returns the ULONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

## -remarks
