# ExtRemoteList::GetNodeOffset method



## -description
<p>The <b>GetNodeOffset</b> method returns the address of the current list item.</p>


## -syntax

````
ULONG64 GetNodeOffset();
````


## -parameters


## -returns
<p><b>GetNodeOffset</b> returns the location, in the target's memory, of the current item for the current list iteration.</p>

<p><b>GetNodeOffset</b> returns the location, in the target's memory, of the current item for the current list iteration.</p>
