# ExtRemoteData::GetLong64 method



## -description
<p>The <b>GetLong64</b> method returns a LONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object, which represents the contents of the target's memory.</p>


## -syntax

````
LONG64 GetLong64();
````


## -parameters


## -returns
<p><b>GetLong64</b> returns the LONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetLong64</b> returns the LONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetLong64</b> returns the LONG64 version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>
