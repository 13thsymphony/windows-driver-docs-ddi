# ExtRemoteList::Prev method



## -description
<p>The <b>Prev</b> method changes the current item to the previous item in the list.</p>


## -syntax

````
void Prev();
````


## -parameters


## -returns
<p>This method does not return a value.</p>

<p>This method does not return a value.</p>

<p>This method does not return a value.</p>

## -remarks
<p>This method can only be used when iterating over doubly-linked lists.</p>

<p>If <b>Prev</b> reaches the end of the list, subsequent calls to <a href="https://msdn.microsoft.com/library/windows/hardware/ff544136">ExtRemoteList::HasNode</a> will return <code>false</code>.</p>
