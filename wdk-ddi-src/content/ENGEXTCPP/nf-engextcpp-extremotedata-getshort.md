# ExtRemoteData::GetShort method



## -description
<p>The <b>GetShort</b> method returns a SHORT version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object, which represents the contents of the target's memory.</p>


## -syntax

````
SHORT GetShort();
````


## -parameters


## -returns
<p><b>GetShort</b> returns the SHORT version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetShort</b> returns the SHORT version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetShort</b> returns the SHORT version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>
