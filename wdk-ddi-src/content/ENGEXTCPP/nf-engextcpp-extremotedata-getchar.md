# ExtRemoteData::GetChar method



## -description
<p>The <b>GetChar</b> method returns a CHAR version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object, which represents the contents of the target's memory.</p>


## -syntax

````
CHAR GetChar();
````


## -parameters


## -returns
<p><b>GetChar</b> returns the CHAR version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetChar</b> returns the CHAR version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>

<p><b>GetChar</b> returns the CHAR version of the <a href="https://msdn.microsoft.com/library/windows/hardware/ff544008">ExtRemoteData</a> object.</p>
