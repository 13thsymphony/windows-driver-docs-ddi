# Reshub.h header


This header is used by SPB. For more information, see
- [SPB](../_SPB/index.md)

Reshub.h contain these programming interfaces:


## Structures

| Title   | Description   |
| ---- |:---- |
| [PNP_SERIAL_BUS_DESCRIPTOR structure](ns-reshub--pnp-serial-bus-descriptor.md) | The PNP_SERIAL_BUS_DESCRIPTOR structure describes the physical connection of a target device to a serial bus (I2C, SPI, or UART). |
| [RH_QUERY_CONNECTION_PROPERTIES_OUTPUT_BUFFER structure](ns-reshub--rh-query-connection-properties-output-buffer.md) | The RH_QUERY_CONNECTION_PROPERTIES_OUTPUT_BUFFER structure contains the connection properties for a target device that is connected to a serial bus (I2C, SPI, or UART). |
