# _T0_DATA structure
The T0_DATA structure is used by the smart card driver library to process T0 I/O.

## Syntax
```
typedef struct _T0_DATA {
  ULONG Lc;
  ULONG Le;
} T0_DATA, *PT0_DATA;
```

## Members


`Lc`

Number of data bytes in the request.

`Le`

Number of expected bytes from the card.
