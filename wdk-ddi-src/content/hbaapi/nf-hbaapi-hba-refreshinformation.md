# HBA_RefreshInformation function



## -description
<p>The <b>HBA_RefreshInformation</b> routine refreshes the library's internally cached data for the indicated HBA.</p>


## -syntax

````
void HBA_API HBA_RefreshInformation(
  _In_ HBA_HANDLE HbaHandle
);
````


## -parameters
<dl>

### -param <i>HbaHandle</i> [in]

<dd>
<p>Contains a value returned by the routine <a href="..\hbaapi\nf-hbaapi-hba-openadapter.md">HBA_OpenAdapter</a> that identifies the HBA whose cached data the library will refresh.</p>
</dd>
</dl>

## -returns
<p>None</p>
