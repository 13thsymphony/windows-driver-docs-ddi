# HBA_FreeLibrary function



## -description
<p>The <b>HBA_FreeLibrary</b> routine releases system resources associated with fibre channel HBA library.</p>


## -syntax

````
HBA_STATUS HBA_API HBA_FreeLibrary(void);
````


## -parameters


## -returns
<p>The <b>HBA_FreeLibrary</b> routine returns a value of type <a href="storage.hba_status">HBA_STATUS</a> that indicates the status of the HBA.</p>

<p>The <b>HBA_FreeLibrary</b> routine returns a value of type <a href="storage.hba_status">HBA_STATUS</a> that indicates the status of the HBA.</p>
