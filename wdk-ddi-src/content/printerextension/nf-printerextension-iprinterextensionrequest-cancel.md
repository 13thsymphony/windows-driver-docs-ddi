# IPrinterExtensionRequest::Cancel method



## -description
Completes the extension event with a cancellation.



## -syntax

````
HRESULT Cancel(
  [in] HRESULT hr,
  [in] BSTR    bstrLogMessage
);
````


## -parameters

### -param hr [in]

The operation result.


### -param bstrLogMessage [in]

The log message.


## -returns
This method returns an <b>HRESULT</b> value.
