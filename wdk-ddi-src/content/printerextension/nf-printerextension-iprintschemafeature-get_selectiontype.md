# get_SelectionType method
Gets the selection type of the Feature.

This property is read-only.

## Syntax

````
HRESULT get_SelectionType(
  [out, retval] PrintSchemaSelectionType *pSelectionType
);
````

## Parameters

`pSelectionType`
