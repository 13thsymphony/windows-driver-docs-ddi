# IPrinterExtensionEvent::OnDriverEvent method
Called when a driver event occurs.

## Syntax

```
HRESULT OnDriverEvent(
  IPrinterExtensionEventArgs *pEventArgs
);
```

## Parameters

`pEventArgs`

The event arguments.


## Return Value

This method returns an <b>HRESULT</b> value.
