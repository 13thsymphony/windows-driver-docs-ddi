# GetOptions method
Gets all the options of a feature.

## Syntax

````
HRESULT GetOptions(
  [in]          IPrintSchemaFeature          *pFeature,
  [out, retval] IPrintSchemaOptionCollection **ppOptions
);
````

## Parameters

`pFeature`

The specified feature.

`ppOptionCollection`




## Return Value

This method returns an <b>HRESULT</b> value.
