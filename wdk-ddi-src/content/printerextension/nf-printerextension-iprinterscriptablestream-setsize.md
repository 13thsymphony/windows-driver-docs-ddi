# IPrinterScriptableStream::SetSize method
Sets the size of the scriptable stream, in bytes.

## Syntax

```
HRESULT SetSize(
  LONG lSize
);
```

## Parameters

`lSize`

The new size in bytes.


## Return Value

This method returns an <b>HRESULT</b> value.
