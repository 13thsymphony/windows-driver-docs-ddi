# IPrintSchemaElement::get_NamespaceUri method
Gets the namespace URI  value of the "name" attribute of this node.

This property is read-only.

## Syntax

```
HRESULT get_NamespaceUri(
  BSTR *pbstrNamespaceUri
);
```

## Parameters

`pbstrNamespaceUri`




## Return Value

None
