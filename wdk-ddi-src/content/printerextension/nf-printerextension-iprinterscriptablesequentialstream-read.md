# Read method
The Read method reads bytes from the stream and returns them as a JavaScript array.

## Syntax

````
HRESULT Read(
  [in]          LONG      cbRead,
  [out, retval] IDispatch **ppdispArray
);
````

## Parameters

`cbRead`

The number of bytes to read.

`ppArray`




## Return Value

This method returns an <b>HRESULT</b> value.
