# IPrinterPropertyBag::GetInt32 method
Reads an integer property.

## Syntax

```
HRESULT GetInt32(
  BSTR bstrName,
  LONG *pnValue
);
```

## Parameters

`bstrName`

The property to read.

`pnValue`

The value read.


## Return Value

This method returns an <b>HRESULT</b> value.
