# get_PageImageableSize method
Gets the imageable area information of the printer.

This property is read-only.

## Syntax

````
HRESULT get_PageImageableSize(
  [out, retval] IPrintSchemaPageImageableSize **ppPageImageableSize
);
````

## Parameters

`ppPageImageableSize`




## Return Value

None
