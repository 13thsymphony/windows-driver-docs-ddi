# IoGetActivityIdThread function



## -description
<p>The IoGetActivityIdThread routine returns the activity ID associated with the current thread.</p>


## -syntax

````
LPCGUID IoGetActivityIdThread(
    None.
);
````

<dl>

### -param <i>None.</i>
