# _WHEA_REVISION structure



## -description
The WHEA_REVISION union describes the revision of the error record data structures.



## -syntax

````
typedef union _WHEA_REVISION {
  struct {
    UCHAR MinorRevision;
    UCHAR MajorRevision;
  };
  USHORT AsUSHORT;
} WHEA_REVISION, *PWHEA_REVISION;
````


## -struct-fields

### -field MinorRevision

The minor revision number.


### -field MajorRevision

The major revision number.


### -field AsUSHORT

A USHORT representation of the contents of the WHEA_REVISION union.


## -remarks
A WHEA_REVISION union is contained within the <a href="whea.whea_error_record_header">WHEA_ERROR_RECORD_HEADER</a> and the <a href="whea.whea_error_record_section_descriptor">WHEA_ERROR_RECORD_SECTION_DESCRIPTOR</a> structures.
