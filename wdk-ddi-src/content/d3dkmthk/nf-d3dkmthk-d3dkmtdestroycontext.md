# D3DKMTDestroyContext function
The <b>D3DKMTDestroyContext</b> function releases a kernel-mode device context.

## Syntax

```
NTSTATUS D3DKMTDestroyContext(
  CONST *D3DKMT_DESTROYCONTEXT
);
```
