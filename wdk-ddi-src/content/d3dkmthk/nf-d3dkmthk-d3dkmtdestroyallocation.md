# D3DKMTDestroyAllocation function



## -description
<p>The <b>D3DKMTDestroyAllocation</b> function releases a resource, a list of allocations, or both.</p>


## -syntax

````
NTSTATUS D3DKMTDestroyAllocation(
  _In_ const D3DKMT_DESTROYALLOCATION *pData
);
````


## -parameters
<dl>

### -param <i>pData</i> [in]

<dd>
<p>A pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff547868">D3DKMT_DESTROYALLOCATION</a> structure that contains information for releasing allocations.</p>
</dd>
</dl>

## -returns
<p><b>D3DKMTDestroyAllocation</b> returns one of the following values:</p><dl>
<dt><b>STATUS_SUCCESS</b></dt>
</dl><p>Allocations were successfully released.</p><dl>
<dt><b>STATUS_INVALID_PARAMETER</b></dt>
</dl><p>Parameters were validated and determined to be incorrect.</p>

<p> </p>

<p>This function might also return other <b>NTSTATUS</b> values.</p>

<p>The following code example demonstrates how an OpenGL ICD can use <b>D3DKMTDestroyAllocation</b> to release a resource and therefore all of the allocations that are associated with it.</p>
