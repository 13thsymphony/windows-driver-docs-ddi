# D3DKMTOpenAdapterFromLuid function
Maps a locally unique identifier (LUID) to a graphics adapter handle.

## Syntax

```
NTSTATUS D3DKMTOpenAdapterFromLuid(
  CONST *D3DKMT_OPENADAPTERFROMLUID
);
```
