# D3DKMTFlipOverlay function
The <b>D3DKMTFlipOverlay</b> function changes the allocation to display on the overlay.

## Syntax

````
NTSTATUS D3DKMTFlipOverlay(
  _In_ const D3DKMT_FLIPOVERLAY *pData
);
````

`D3DKMT_FLIPOVERLAY`

<th>Description</th>
