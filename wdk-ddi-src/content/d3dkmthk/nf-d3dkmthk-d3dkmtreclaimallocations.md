# D3DKMTReclaimAllocations function
Reclaims video memory allocations.

## Syntax

````
EXTERN_C _Check_return_ NTSTATUS APIENTRY D3DKMTReclaimAllocations(
  _Inout_ const D3DKMT_RECLAIMALLOCATIONS *pData
);
````
