# D3DKMTDestroyKeyedMutex function
The <b>D3DKMTDestroyKeyedMutex</b> function destroys a keyed mutex object.

## Syntax

````
NTSTATUS D3DKMTDestroyKeyedMutex(
  _In_ const D3DKMT_DESTROYKEYEDMUTEX *pData
);
````
