# D3DKMTDestroyDCFromMemory function
The <b>D3DKMTDestroyDCFromMemory</b> function releases the display context.

## Syntax

````
NTSTATUS D3DKMTDestroyDCFromMemory(
  _In_ const D3DKMT_DESTROYDCFROMMEMORY *pData
);
````
