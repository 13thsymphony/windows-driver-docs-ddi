# KsAcquireDevice function



## -description
<p>The<b> KsAcquireDevice</b> function gains synchronous access for <i>Device</i> by acquiring the device mutex.</p>


## -syntax

````
void KsAcquireDevice(
  _In_ PKSDEVICE Device
);
````


## -parameters
<dl>

### -param Device [in]

<dd>
<p>An AVStream device for which synchronous control should be acquired.</p>
</dd>
</dl>

## -returns
<p>None</p>

## -remarks
<p>For more information, see <a href="https://msdn.microsoft.com/011edaaa-7449-41c3-8cfb-0d319901af8b">Mutexes in AVStream</a>.</p>
