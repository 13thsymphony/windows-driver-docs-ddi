# _D3DDDI_MULTIPLANE_OVERLAY_FLAGS Enumeration
Identifies a flip operation to be performed on an overlay plane.

## Syntax
```
typedef enum _D3DDDI_MULTIPLANE_OVERLAY_FLAGS {
  D3DDDI_MULTIPLANE_OVERLAY_FLAG_VERTICAL_FLIP    ,
  D3DDDI_MULTIPLANE_OVERLAY_FLAG_HORIZONTAL_FLIP
} D3DDDI_MULTIPLANE_OVERLAY_FLAGS;
```

## Constants

<table>
            
                <tr>
                    <td>D3DDDI_MULTIPLANE_OVERLAY_FLAG_VERTICAL_FLIP</td>
                    <td>The overlay plane should flip the data vertically, making it appear upside-down.</td>
                </tr>
            
                <tr>
                    <td>D3DDDI_MULTIPLANE_OVERLAY_FLAG_HORIZONTAL_FLIP</td>
                    <td>The overlay plane should flip the data horizontally, making it appear as a right-to-left mirror image.</td>
                </tr>
</table>
