# PFND3DDDI_MULTIPLYTRANSFORM callback



## -description
<p>The <i>MultiplyTransform</i> function modifies the current transform.</p>


## -prototype

````
PFND3DDDI_MULTIPLYTRANSFORM MultiplyTransform;

__checkReturn HRESULT APIENTRY MultiplyTransform(
  _In_       HANDLE                      hDevice,
  _In_ const D3DDDIARG_MULTIPLYTRANSFORM *pData
)
{ ... }
````


## -parameters
<dl>

### -param <i>hDevice</i> [in]

<dd>
<p> A handle to the display device (graphics context).</p>
</dd>

### -param <i>pData</i> [in]

<dd>
<p> A pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/ff543221">D3DDDIARG_MULTIPLYTRANSFORM</a> structure that describes how to modify the current transform.</p>
</dd>
</dl>

## -returns
<p><i>MultiplyTransform</i> returns S_OK or an appropriate error result if the transform is not successfully modified.</p>
