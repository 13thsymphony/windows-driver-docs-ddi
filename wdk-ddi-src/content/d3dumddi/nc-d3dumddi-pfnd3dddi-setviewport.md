# PFND3DDDI_SETVIEWPORT callback



## -description
<p>The <i>SetViewport</i> function informs guard-band-aware drivers of the view-clipping rectangle.</p>


## -prototype

````
PFND3DDDI_SETVIEWPORT SetViewport;

__checkReturn HRESULT APIENTRY SetViewport(
  _In_       HANDLE                 hDevice,
  _In_ const D3DDDIARG_VIEWPORTINFO *pData
)
{ ... }
````


## -parameters
<dl>

### -param <i>hDevice</i> [in]

<dd>
<p> A handle to the display device (graphics context).</p>
</dd>

### -param <i>pData</i> [in]

<dd>
<p> A pointer to a <a href="..\d3dumddi\ns-d3dumddi--d3dddiarg-viewportinfo.md">D3DDDIARG_VIEWPORTINFO</a> structure that describes the view-clipping rectangle.</p>
</dd>
</dl>

## -returns
<p><i>SetViewport</i> returns S_OK or an appropriate error result if the driver is not successfully informed about the view-clipping rectangle.</p>
