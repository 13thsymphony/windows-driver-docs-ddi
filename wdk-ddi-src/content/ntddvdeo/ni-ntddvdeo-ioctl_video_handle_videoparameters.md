# IOCTL_VIDEO_HANDLE_VIDEOPARAMETERS IOCTL



## -description

Gets the capabilities of the device's television connector and/or copy protection hardware, or sets the desired functionality on the copy protection hardware. Adapters that support TV connectors should provide support for this modal IOCTL. See <a href="https://msdn.microsoft.com/7d7d44b5-3248-4bee-bc4d-e02fd3c606a7">TV Connector and Copy Protection Support in Video Miniport Drivers</a> for details.

### -input-buffer
The VRP <b>InputBuffer</b> contains a <a href="display.videoparameters">VIDEOPARAMETERS</a> structure that describes the action to be performed by the miniport driver.

### -status-block
I/O Status block
If the miniport driver successfully gets or sets the requested information, it sets the <b>Information</b> member of the VRP's <a href="display.status_block">STATUS_BLOCK</a> to <b>sizeof</b>(VIDEOPARAMETERS); otherwise, the miniport driver sets this member to zero.
