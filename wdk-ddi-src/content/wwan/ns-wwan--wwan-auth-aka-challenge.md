# WWAN_AUTH_AKA_CHALLENGE structure



## -description
<p>The WWAN_AUTH_AKA_CHALLENGE structure represents an authentication challenge using the AKA method.</p>


## -syntax

````
typedef struct _WWAN_AUTH_AKA_CHALLENGE {
  BYTE Rand[WWAN_AUTH_RAND_LEN];
  BYTE Autn[WWAN_AUTH_AUTN_LEN];
} WWAN_AUTH_AKA_CHALLENGE, *PWWAN_AUTH_AKA_CHALLENGE;
````


## -struct-fields
<dl>

### -field <b>Rand[WWAN_AUTH_RAND_LEN]</b>

<dd>
<p>128-bit random number challenge. This member represents a multi-byte value in little-endian format.</p>
</dd>

### -field <b>Autn[WWAN_AUTH_AUTN_LEN]</b>

<dd>
<p>128-bit authentication value generated by the AuC, which, together  with the value of the <b>Rand</b> member authenticates the server to the peer. This member represents a multi-byte value in little-endian format.</p>
</dd>
</dl>
