# DOT11_SSID structure



## -description

## -syntax

````
typedef struct _DOT11_SSID {
  ULONG uSSIDLength;
  UCHAR ucSSID[DOT11_SSID_MAX_LENGTH];
} DOT11_SSID, *PDOT11_SSID;
````


## -struct-fields
<dl>

### -field <b>uSSIDLength</b>

<dd>
<p>The length, in bytes, of the 
     <b>ucSSID</b> array.</p>
</dd>

### -field <b>ucSSID</b>

<dd>
<p>The SSID.</p>
</dd>
</dl>

## -remarks
<p>The SSID that is specified by the 
    <b>ucSSID</b> member is not a null-terminated ASCII string. The length of the SSID is determined by the 
    <b>uSSIDLength</b> member.</p>

<p>A 
    <i>wildcard SSID</i> is an SSID whose 
    <b>uSSIDLength</b> member is set to zero. When the desired SSID is set to the wildcard SSID, the 802.11
    station can connect to any basic service set (BSS) network.</p>
