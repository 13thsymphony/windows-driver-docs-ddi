# RtlUpcaseUnicodeChar function



## -description
<p>The <b>RtlUpcaseUnicodeChar</b> routine converts the specified Unicode character to uppercase.</p>


## -syntax

````
WCHAR RtlUpcaseUnicodeChar(
  _In_ WCHAR SourceCharacter
);
````


## -parameters
<dl>

### -param SourceCharacter [in]

<dd>
<p>Specifies the character to convert.</p>
</dd>
</dl>

## -returns
<p><b>RtlUpcaseUnicodeChar</b> returns the uppercase version of the specified Unicode character.</p>
