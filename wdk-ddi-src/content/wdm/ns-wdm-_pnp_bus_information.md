# _PNP_BUS_INFORMATION structure
The <b>PNP_BUS_INFORMATION</b> structure describes a bus.

## Syntax
````
typedef struct _PNP_BUS_INFORMATION {
  GUID           BusTypeGuid;
  INTERFACE_TYPE LegacyBusType;
  INTERFACE_TYPE LegacyBusType;
  ULONG          BusNumber;
````
