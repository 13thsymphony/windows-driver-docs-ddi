# RtlFindLeastSignificantBit function



## -description
<p>The <b>RtlFindLeastSignificantBit</b> routine returns the zero-based position of the least significant nonzero bit in its parameter.</p>


## -syntax

````
CCHAR RtlFindLeastSignificantBit(
  _In_ ULONGLONG Set
);
````


## -parameters
<dl>

### -param <i>Set</i> [in]

<dd>
<p>The 64-bit value to be searched for its least significant nonzero bit. </p>
</dd>
</dl>

## -returns
<p>The zero-based bit position of the least significant nonzero bit, or -1 if every bit is zero.</p>
