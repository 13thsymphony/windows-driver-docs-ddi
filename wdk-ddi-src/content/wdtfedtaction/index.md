This header is used by dtf. For more information, see
- [dtf](../_dtf/index.md)

Wdtfedtaction.h contain these programming interfaces:


## Interfaces

| Title   | Description   |
| ---- |:---- |
| [IWDTFEnhancedDeviceTestSupportAction2 interface](nn-wdtfedtaction-iwdtfenhanceddevicetestsupportaction2.md) | Defines operations and properties that support the Enhanced Device Test (EDT) filter driver. |
