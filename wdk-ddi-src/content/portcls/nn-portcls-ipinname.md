# IPinName interface



## -description
<p>In Windows 7 and later operating systems, the <code>IPinName</code> interface is used by miniport drivers to report and update the names of audio endpoints.</p>


## -inheritance
<p>The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IPinName</b> interface inherits from the <a href="com.iunknown" xmlns:loc="http://microsoft.com/wdcml/l10n"><b>IUnknown</b></a> interface but does not have additional members.</p>
