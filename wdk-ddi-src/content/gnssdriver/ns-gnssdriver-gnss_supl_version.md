# GNSS_SUPL_VERSION structure
This structure contains SUPL version information.

## Syntax
````
typedef struct {
  ULONG MajorVersion;
  ULONG MinorVersion;
} GNSS_SUPL_VERSION, *PGNSS_SUPL_VERSION;
````

## Members

        
            `MajorVersion`

            Major version number.
        
            `MinorVersion`

            Minor version number.
