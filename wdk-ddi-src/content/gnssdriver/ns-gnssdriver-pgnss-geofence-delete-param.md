# PGNSS_GEOFENCE_DELETE_PARAM structure



## -description
<p>This structure is used for deleting a geofence.</p>


## -syntax

````
typedef struct {
  ULONG Size;
  ULONG Version;
  ULONG GeofenceID;
  BYTE Â Unused[512];
} GNSS_GEOFENCE_DELETE_PARAM, *PGNSS_GEOFENCE_DELETE_PARAM;
````


## -struct-fields
<dl>

### -field <b>Size</b>

<dd>
<p>Structure size.</p>
</dd>

### -field <b>Version</b>

<dd>
<p>Version number.</p>
</dd>

### -field <b>GeofenceID</b>

<dd>
<p>The ID of the geofence.</p>
<p> This ID was generated by the GNSS engine during creation of the geofence. Once deleted, the ID can be recycled by the GNSS engine and assigned to a newly created geofence.</p>
</dd>

### -field <b>Unused[512]</b>

<dd>
<p>Padding buffer.</p>
</dd>
</dl>
