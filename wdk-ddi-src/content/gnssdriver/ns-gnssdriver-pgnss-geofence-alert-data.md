# PGNSS_GEOFENCE_ALERT_DATA structure



## -description
<p>This structure is used by the GNSS engine to notify a geofence breach alert.</p>


## -syntax

````
typedef struct {
  ULONG                 Size;
  ULONG                 Version;
  ULONG                 GeofenceID;
  GNSS_GEOFENCE_STATE   GeofenceState;
  GNSS_FIXDATA_BASIC    FixBasicData;
  GNSS_FIXDATA_ACCURACY FixAccuracyData;
  BYTE                  Unused[512];
} GNSS_GEOFENCE_ALERT_DATA, *PGNSS_GEOFENCE_ALERT_DATA;
````


## -struct-fields
<dl>

### -field <b>Size</b>

<dd>
<p>Structure size.</p>
</dd>

### -field <b>Version</b>

<dd>
<p>Version number.</p>
</dd>

### -field <b>GeofenceID</b>

<dd>
<p>The ID of the geofence. This ID was generated by the GNSS engine during creation of the geofence.</p>
</dd>

### -field <b>GeofenceState</b>

<dd>
<p>The new state of the geofence. The alert implies transitioning to this state.</p>
</dd>

### -field <b>FixBasicData</b>

<dd>
<p>The fix used to determine the geofence breach. Instead of the full set of fix data, a smaller subset contained in this field and the  FixAccuracyData field is expected.</p>
</dd>

### -field <b>FixAccuracyData</b>

<dd>
<p>The fix used to determine the geofence breach. Instead of the full set of fix data, a smaller subset contained in this field and the  FixBasicData field is expected.</p>
</dd>

### -field <b>Unused[512]</b>

<dd>
<p>Padding buffer.</p>
</dd>
</dl>
