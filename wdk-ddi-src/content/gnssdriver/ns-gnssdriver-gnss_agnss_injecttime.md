# GNSS_AGNSS_INJECTTIME structure
This structure defines the format for AGNSS time injection.

## Syntax
````
typedef struct {
  ULONG    Size;
  ULONG    Version;
  FILETIME UtcTime;
  ULONG    TimeUncertainty;
} GNSS_AGNSS_INJECTTIME, *PGNSS_AGNSS_INJECTTIME;
````

## Members


`Size`

Structure size.

`TimeUncertainty`

Uncertainty in milliseconds.

`UtcTime`

UTC time.

`Version`

Version number.
