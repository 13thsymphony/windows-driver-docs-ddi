# get_Config method
Gets the WDTF configuration object.

This property is read-only.

## Syntax

````
HRESULT get_Config(
  [out, retval] IWDTFCONFIG2 **ppConfig
);
````

## Parameters

`ppConfig`




## Return Value

None
