# get_WDTF method
Gets the main WDTF aggregation object.

This property is read-only.

## Syntax

````
HRESULT get_WDTF(
  [out, retval] IWDTF2 **ppWDTF
);
````

## Parameters

`ppWDTF`




## Return Value

None
