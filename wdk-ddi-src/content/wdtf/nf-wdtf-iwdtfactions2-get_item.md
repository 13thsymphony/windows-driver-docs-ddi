# get_Item method
Gets an individual device in the DeviceDepot.

This property is read-only.

## Syntax

````
HRESULT get_Item(
  [in]          LONG         Index,
  [out, retval] IWDTFTarget2 **ppTarget
);
````

`ppAction`
