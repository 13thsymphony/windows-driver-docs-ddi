# IWDTFSystemAction2::get_Critical method
Gets or sets a value that indicates whether the the system power state is critical.

This property is read/write.

## Syntax

```
HRESULT get_Critical(
  VARIANT_BOOL *pbCritical
);
```

## Parameters

`pbCritical`




## Return Value

None
