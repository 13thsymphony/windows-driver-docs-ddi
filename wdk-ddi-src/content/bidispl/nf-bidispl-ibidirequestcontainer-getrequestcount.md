# GetRequestCount method
The <b>GetRequestCount</b> method gets the number of requests in the request list.

## Syntax

````
HRESULT GetRequestCount(
  [out] ULONG *puCount
);
````

## Parameters

`puCount`

Pointer to a variable that receives the number of requests.
