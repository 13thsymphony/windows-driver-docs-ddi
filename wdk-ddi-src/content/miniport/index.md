# miniport.h header



miniport.h contains the following programming interfaces:







## Structures
| Title | Description |
| ---- |:---- |
| [_EMULATOR_ACCESS_ENTRY](ns-miniport-_emulator_access_entry.md) | The EMULATOR_ACCESS_ENTRY structure specifies an element in the EmulatorAccessEntries array set up in the VIDEO_PORT_CONFIG_INFO structure by drivers of VGA-compatible (SVGA) adapters on x86-based NT-based operating system platforms. |
| [_GROUP_AFFINITY](ns-miniport-_group_affinity.md) | The GROUP_AFFINITY structure specifies a group number and the processor affinity within that group. |
| [_PROCESSOR_NUMBER](ns-miniport-_processor_number.md) | The PROCESSOR_NUMBER structure identifies a processor by its group number and group-relative processor number. |
