# PFND3DDDI_BUFBLT1 callback



## -description
<p>Performs a bit-block transfer (bitblt) operation from a source vertex or index buffer to a destination vertex or index buffer. Implemented by Windows Display Driver Model (WDDM) 1.2 or later user-mode display drivers.</p>


## -prototype

````
PFND3DDDI_BUFBLT1 BufBlt1;

__checkReturn HRESULT APIENTRY* BufBlt1(
  _In_       HANDLE               hDevice,
  _In_ const D3DDDIARG_BUFFERBLT1 *pData
)
{ ... }
````


## -parameters
<dl>

### -param <i>hDevice</i> [in]

<dd>
<p> A handle to the display device (graphics context).</p>
</dd>

### -param <i>pData</i> [in]

<dd>
<p> A pointer to a <a href="https://msdn.microsoft.com/library/windows/hardware/hh451069">D3DDDIARG_BUFFERBLT1</a> structure that describes the parameters of the buffer bitblt operation.</p>
</dd>
</dl>

## -returns
<p>
      Returns S_OK or an appropriate error result if the buffer bitblt operation is not successfully performed.</p>
