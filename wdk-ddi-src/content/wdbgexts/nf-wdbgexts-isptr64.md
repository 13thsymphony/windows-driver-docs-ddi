# IsPtr64 function



## -description
<p>The <b>IsPtr64</b> function determines if the target uses 64-bit pointers.</p>


## -syntax

````
ULONG IsPtr64(void);
````


## -parameters


## -returns
<p>The function returns <b>TRUE</b> if the target uses 64-bit pointers; <b>FALSE</b>, otherwise.</p>
