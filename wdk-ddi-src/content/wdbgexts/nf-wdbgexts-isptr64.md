# IsPtr64 function
The <b>IsPtr64</b> function determines if the target uses 64-bit pointers.

## Syntax

````
ULONG IsPtr64(void);
````

## Parameters

This function has no parameters.

## Return Value

The function returns <b>TRUE</b> if the target uses 64-bit pointers; <b>FALSE</b>, otherwise.
