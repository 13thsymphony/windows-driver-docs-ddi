# IKsPinEx interface



## -description
<p>The <b>IKsPinEx</b> interface inherits all the methods of the <b>IKsPin</b> interface and extends <b>IKsPin</b> to provide a method that notifies the filter graph of an error to give the filter graph an opportunity to halt.</p>
<p>The IID for this interface is IID_IKsPinEx.</p>


## -inheritance
<p>The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IKsPinEx</b> interface inherits from <a href="https://msdn.microsoft.com/library/windows/hardware/ff559896">IKsPin</a>. <b>IKsPinEx</b> also has these types of members:</p>
