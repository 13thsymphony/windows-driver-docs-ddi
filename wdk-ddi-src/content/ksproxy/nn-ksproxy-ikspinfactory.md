# IKsPinFactory interface



## -description
<p>The <b>IKsPinFactory</b> interface provides a method that retrieves the identifier of a pin factory.</p>


## -inheritance
<p>The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IKsPinFactory</b> interface inherits from the <a href="com.iunknown" xmlns:loc="http://microsoft.com/wdcml/l10n"><b>IUnknown</b></a> interface. <b>IKsPinFactory</b> also has these types of members:</p>

<p>The <b>IKsPinFactory</b> interface has these methods.</p>

<p>Retrieves the identifier of a pin factory.</p>

<p> </p>

<p>The <b xmlns:loc="http://microsoft.com/wdcml/l10n">IKsPinFactory</b> interface inherits from the <a href="com.iunknown" xmlns:loc="http://microsoft.com/wdcml/l10n"><b>IUnknown</b></a> interface. <b>IKsPinFactory</b> also has these types of members:</p>
