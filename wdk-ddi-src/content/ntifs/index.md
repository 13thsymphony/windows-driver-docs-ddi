# Ntifs.h header

## -description

This header is used by Installable file system, Windows kernel, NetVista. For more information, see
- [Installable file system](../_ifsk/index.md)
- [Windows kernel](../_kernel/index.md)
- [NetVista](../_NetVista/index.md)
