# IOCTL_BTHHFP_DEVICE_GET_KSNODETYPES IOCTL



## -description
The <b>IOCTL_BTHHFP_DEVICE_GET_KSNODETYPES</b> 
   IOCTL Gets the KSNODE types that best describe the Bluetooth device’s input and output.
